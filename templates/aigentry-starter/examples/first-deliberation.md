# Your First Deliberation

## 1. Start a Discussion
In Claude Code, use the MCP tool:
```
deliberation_start with topic "Should we use REST or GraphQL for our API?"
```

## 2. Add Perspectives
```
deliberation_respond with your analysis of trade-offs
```

## 3. Get Consensus
```
deliberation_synthesize to generate a summary with verdict
```

## Tips
- Keep topics focused on a single decision
- Provide concrete constraints (timeline, team size, existing tech)
- Use personas for diverse perspectives (see context/deliberation-personas.json)
