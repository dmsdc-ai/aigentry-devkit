위 6개 단서를 바탕으로 다음 3개 sub-question을 모두 답하고, **추론 과정**도 함께 보여줘.

(Q-A) **누가 Python을 쓰나?**

(Q-B) **Rust 사용자의 모니터 사이즈(인치)는?**

(Q-C) **43" 모니터는 몇 번 데스크에 있나?**

출력 형식:

```
## 추론
- step 1: …
- step 2: …
- step 3: …
- step 4: …
- (필요한 만큼 더)

## 답
- (Q-A) 답: …
- (Q-B) 답: …
- (Q-C) 답: …
```

규칙:

(1) 답 3개 모두 정확해야 만점. 부분 정답도 부분 점수.

(2) 추론 단계는 최소 3개 이상 — 단순히 "답: X" 만 적으면 추론 점수 0점.

(3) 단서 번호(예: "단서 3에 의해 …", "clue 5 + clue 2 → …")로 단계마다 어떤 단서를 썼는지 표시.

(4) 가정 금지 — 단서에 명시되지 않은 사실(예: "Alice는 보통 Python 좋아하지" 같은 외부 지식)을 가져오지 마.

(5) 추측 금지. 한 단서로 다중 가능성이 남으면 다음 단서를 써서 좁혀라.
