위 한국어 launch 브리핑을 **영문 외부 발표용 자료**로 번역해줘. 다음 보존 규칙을 지켜야 한다.

(P1) **고유명사 보존** — 사람/제품/지역 고유명사는 한글 그대로 유지하지 말고 brief에 명시된 공식 영문 표기를 그대로 써. 예: "박서연" → "Park Seoyeon", "한국지사" → "Korea office", "제품명 노바3" → "Nova 3".

(P2) **수치 / 버전 / API 이름 그대로** — `v3.7.2`, `p99 < 80ms`, `2026-06-15`, `4,200 RPS`, `/v1/orders` 같은 토큰은 변형 없이 그대로 유지.

(P3) **문서 구조 보존** — 한국어 원문의 `## 헤더`, `### 서브헤더`, bullet list, 표(table)를 모두 1:1로 영문판에 유지. 헤더 개수가 같아야 함.

(P4) **약어 첫 등장 시 풀어쓰기** — 한국어 원문에 풀어 표기된 약어(예: "PSP — Payment Service Provider")는 영문판에서도 첫 등장 시 풀어쓰기 + 약어 형식 유지.

(P5) **단순 의역 금지 + 가짜 정보 금지** — 원문에 없는 사실(예: "Nova 3는 시장 1위")을 추가하지 마. 원문 내용만 영어로 옮김.

출력은 영어 번역 본문만. 별도 메타 코멘트 / 자기 점검 메모 / "다음과 같이 번역했습니다"류 prologue 쓰지 마.
